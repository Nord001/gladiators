                     Поисковый Движок для Web
                 WFSearch .PRO версия 1.0 (для PHP)

  - быстрый, индексированный поиск
  - индексация по содержанию файлов
  - использование базы данных MySQL
  - рекурсивная индексация в подкаталогах
  - разрешенные и запрещенные для поиска файлы
  - постраничный вывод
  - логика поиска "и" / "или"
  - сортировка результатов по популярности
  - уничтожение тэгов
  - шаблоны header и footer
  - многоязыковой интерфейс
  - время показа процесса поиска

   Поисковый движок был написан с нуля, с использованием оригинальной документации по
PHP и MySQL. Как обычно, в интернете я не нашел поисковых движков, удволетворяющих
моим потребностям. И пришлось написать все самостоятельно. Писался движок вцелом в течении
12 дней разработки (или 30 часов).
   Все, который использует этот сценарий являются бета-тестерами. Так что сообщайте мне
обо всех найденых багах и способах их устранения (по возможности). В таком случае, высылайте
мне исходный код с Вашими изменениями, основанный на последней версиии WFSearch .PRO engine.


                          Как установить
                          ~~~~~~~~~~~~~~
   Настройка - очень простая. Пользователи Unix должны установить права на
запуск. Также Вы должны скорректировать файл присоединения к MySQL (connect.php).
Вы можете легко настроить интерфейс (файлы include/header.php и footer.php).
Первый запуск - admin.php. Далее выбрав язык, следуйте инструкциям. Поиск
будет производиться ТОЛЬКО после полной индексации.
